# Occluded Object Detection for Autonomous Vehicles using YOLOv5

This repository contains the code of occluded object detection using the YOLOv5 model on a novel dataset prepared by us.

* **Link to the paper published in IEEE on an updated version of the code and dataset:** https://ieeexplore.ieee.org/abstract/document/9946565
* **The complete dataset is hosted on Roboflow Universe, where it can be exported in multiple object detection formats (e.g., YOLO, COCO, Pascal VOC) using this link:** https://universe.roboflow.com/tanzim-mostafa/p2_dhaka_dataset-f6ba6
* **Direct link to this code's google colab file:** https://colab.research.google.com/drive/1QrkAXB25sCb905ci1JeDLH_0FuLh7kvZ?usp=sharing
